# 📚 AI PDF Summarizer

## 📖 Project Overview

This project is an AI-powered PDF Summarizer built using Python and Streamlit. It allows users to upload a PDF document and automatically generates:
- Short Summary
- Detailed Summary
- Bullet Point Summary

## 🚀 Features

- Upload PDF files
- AI-powered summarization using Groq API
- Short Summary
- Detailed Summary
- Bullet Point Summary
- User-friendly Streamlit interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Groq API
- PyPDF
- python-dotenv

## ▶️ How to Run

1. Install Python.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Create a `.env` file and add:

```text
GROQ_API_KEY=your_api_key_here
```

4. Run:

```bash
python -m streamlit run app.py
```
